# Notification System

## Overview
A React-based Notification Management System built using Vite and Material UI.

## Features
- Display notifications
- Filter notifications by category
- Pagination support
- Unread notification count
- Responsive UI

## Technologies Used
- React
- Vite
- Material UI
- JavaScript

## Project Structure

src/
├── api/
├── components/
├── hooks/
├── pages/
├── App.jsx
└── main.jsx

## Installation

```bash
npm install
npm run dev
```

## Running the Project

```bash
npm run dev
```

Application runs at: http://localhost:5173
